Autos del Litoral
Descripción

Autos del Litoral es un sistema de gestión de concesionaria desarrollado en Python como trabajo final de Programación I.

El programa permite administrar el stock de vehículos mediante una interfaz de consola, almacenando la información en archivos JSON para garantizar la persistencia de los datos.

Funcionalidades
Gestión de Autos
Alta de vehículos.
Listado completo de autos en stock.
Búsqueda de autos por ID o patente.
Modificación de datos de un vehículo.
Cambio de estado (Disponible, Reservado, Vendido, etc.).
Baja de vehículos.
Registro de fecha de alta, modificación y baja.
Control de kilometraje.
Identificación única mediante ID automático.
Gestión de Datos
Almacenamiento en formato JSON.
Lectura y escritura automática de datos.
Conservación de la información entre ejecuciones.
Tecnologías Utilizadas
Python 3
JSON
Programación modular
VS Code
Estructura General

El sistema está dividido en módulos para facilitar su mantenimiento y organización:

main.py
menu.py
autos.py
datos.py
validaciones.py
Objetivo

Desarrollar una aplicación que permita administrar de manera sencilla y eficiente el stock de una concesionaria de automóviles, aplicando los conceptos aprendidos durante la cursada de Programación I.
